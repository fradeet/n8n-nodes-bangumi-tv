# n8n-nodes-bangumi-tv

This is an n8n community node. It lets you use Bangumi.tv in your n8n workflows.

Bangumi.tv is a media tracking and discovery platform centered on anime, comics, games, music, and books. Users also can view or create user-facing data such as anime collections and reviews.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/sustainable-use-license/) workflow automation platform.

## Operations

Avaliable:

- **Subject**: Search, get subject, image, related subjects, characters and persons.
- **Character**: Search, get character, related persons and subjects.
- **Person**: Get person, related characters and subjects.
- **User**: Get user and me info.

## Credentials

The credentials are optional in most operations. But you can authenticate to get full experience.

To authenticate, you need to register a new application on the [Bangumi.tv Developer Platform](https://bangumi.tv/dev), read the [Oauth2.md](docs/user-guide/oauth2.md) to get more information.
