  n8n 声明式节点的 routing 支持三种模式：
  1. 简单 GET - routing: { request: { method: 'GET', url: 
  '=/v0/subjects/{{$value}}' } }

  3. POST/PUT/PATCH - 使用 preSend 函数构建请求体
